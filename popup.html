Answer Saver

Active

Fetches your previous quiz submissions from the Canvas API, finds your highest-scoring answers, and auto-fills them when you hover over each question. Works with multiple choice, true/false, fill-in-the-blank, matching, numerical, and essay questions.

Privacy Guard

Active

Overrides browser visibility and focus APIs so Canvas always thinks your tab is active and focused. Blocks focus, blur, visibilitychange, mouseleave, pagehide, freeze/resume and other tracking events from firing. Also intercepts dispatchEvent to block programmatic tracking.

Kiosk Spoof

Standby

Wraps the current page in an iframe with a fake Canvas Kiosk App top bar, making it look like you're running the locked-down kiosk browser. Activate on any quiz page using the kiosk toggle. Works independently of other features.

AI Answer Engine

Tap to configure

Connects to your own LLM API (OpenAI, Anthropic, Ollama, any OpenAI-compatible endpoint). Three modes: Hover highlights the correct answer, Right-click shows a floating answer box, Keybind auto-types the answer character by character. Click this card to set up.
